# 📖 Cómo Usar PDF Creator Pro

## 🎯 Guía Rápida de Uso

### 1. **Primera Vez - Registro**

1. Abre la aplicación
2. Click en **"Ingresar"** (arriba derecha)
3. Ve a la tab **"Registrarse"**
4. Rellena:
   - Nombre (opcional)
   - Email
   - Contraseña
5. Click **"Registrarse"**
6. ¡Listo! Ya estás dentro 🎉

---

### 2. **Crear tu Primer PDF**

#### Opción A: Desde Plantilla

1. **Navbar** → Click en **"Plantillas"**
2. Verás **2 plantillas**:
   - 🧬 **A&T BioNano** - Brochure corporativo completo
   - 💼 **Business Proposal** - Propuesta de negocios
3. Click en **"Usar Plantilla"**
4. Se abre el **Editor** con las secciones pre-cargadas
5. ¡Ahora puedes editar!

#### Opción B: Desde Cero

1. **Navbar** → Click en **"Crear PDF"**
2. Verás una pantalla vacía
3. Click en **"Abrir Biblioteca de Secciones"**
4. Selecciona secciones una por una
5. Construye tu PDF como bloques de Lego

---

### 3. **Editar Texto** ✍️

#### Click para Editar:
1. **Haz click** en cualquier texto del PDF
2. Aparece un **borde verde** → Está seleccionado
3. **Escribe** directamente
4. Los cambios se ven **inmediatamente**

#### Cambiar Estilo:
Cuando seleccionas texto, aparece una **barra flotante** arriba:

```
┌──────────────────────────────────────┐
│ [16px ▼] [Poppins ▼] [B] [🎨]       │
└──────────────────────────────────────┘
```

- **16px** → Tamaño de fuente (12 a 64px)
- **Poppins** → Tipografía (8 fuentes disponibles)
- **B** → Negrita ON/OFF
- **🎨** → Selector de color

#### Cambiar Color:
1. Click en el ícono **🎨**
2. Aparece un **color picker ChromePicker profesional**
3. Puedes usar:
   - Selector visual de color con paleta completa
   - RGB (Red, Green, Blue)
   - HEX (#FFFFFF)
   - HSL (Hue, Saturation, Lightness)
   - Presets de colores corporativos
4. El color cambia **en tiempo real**

#### Editor de Texto Enriquecido:
1. **Selecciona texto** para activar controles avanzados
2. **Controles disponibles**:
   - Tamaño de fuente: 12px a 64px
   - Tipografía: Poppins, Roboto, Arial, Georgia, etc.
   - Negrita, cursiva, subrayado
   - Color de texto y fondo
   - Alineación: izquierda, centro, derecha
3. **Panel de propiedades**: Configuración avanzada de estilos y apariencia

#### Controles de Zoom:
1. Usa el **zoom** para edición precisa
2. **Vista de canvas** con zoom para detalles finos
3. **Cuadrícula de ayuda** para alineación perfecta

#### Gestor de Capas:
1. **Control de orden** de secciones
2. **Visibilidad** de capas
3. **Organización** jerárquica de elementos

---

### 4. **Agregar Secciones** ➕

1. Click en **"Agregar Sección"** (arriba) o el botón flotante **+** (abajo derecha)
2. Se abre un **Modal hermoso** con la biblioteca
3. **Tabs arriba** para filtrar:
   - Todas
   - Hero
   - About
   - Services
   - Contact
   - Content
   - Values
4. **Click en cualquier card** → La sección se agrega al final de tu PDF
5. El modal se cierra automáticamente

#### Secciones Disponibles:

| Emoji | Nombre | Descripción |
|-------|--------|-------------|
| 🧬 | Hero - A&T BioNano | Header con gradiente verde |
| 👥 | Quiénes Somos | Texto + Imagen lateral |
| 🎯 | Misión y Visión | 2 tarjetas lado a lado |
| 💎 | Valores Corporativos | Grid de 4 valores |
| ⚙️ | Servicios | Grid de servicios con hover |
| 📧 | Contacto | Footer con info de contacto |
| 📝 | Texto Simple | Párrafo editable |
| 🖼️ | Imagen + Texto | Layout 50/50 |

---

### 5. **Reordenar Secciones** 🔀

#### Drag & Drop:
1. **Pasa el mouse** sobre una sección
2. Aparece un **borde verde** y botones arriba-derecha
3. **Arrastra** desde cualquier parte de la sección
4. **Suelta** en la posición deseada
5. Las secciones se reordenan **instantáneamente**

```
Antes:              Después:
┌─────────┐         ┌─────────┐
│ Hero    │         │ About   │ ← Moviste esto arriba
├─────────┤         ├─────────┤
│ About   │ ←──┐    │ Hero    │
├─────────┤    │    ├─────────┤
│ Contact │    └──  │ Contact │
└─────────┘         └─────────┘
```

---

### 6. **Eliminar Sección** 🗑️

1. **Pasa el mouse** sobre la sección
2. Aparecen botones arriba-derecha:
   - **☰** Drag handle (para arrastrar)
   - **🗑️** Delete (para eliminar)
3. Click en **🗑️**
4. La sección se elimina inmediatamente
5. Recibes una notificación: "Sección eliminada"

---

### 7. **Guardar tu PDF** 💾

1. Click en **"Guardar"** (arriba derecha)
2. Se abre un **Dialog**
3. Escribe el **nombre** del documento (ej: "Mi Brochure 2026")
4. Ves un resumen:
   - 📊 Cantidad de secciones
   - 👤 Tu email
5. Click **"Guardar"**
6. El documento se guarda en **"Mis PDFs"**
7. Notificación: "Documento guardado exitosamente" ✅

---

### 8. **Exportar a PDF** 📥

1. Click en **"Exportar PDF"** (arriba derecha)
2. Ves una notificación: "Generando PDF... Esto puede tomar unos segundos"
3. Se genera un archivo `.pdf` profesional
4. Se descarga automáticamente con el nombre de tu documento
5. Notificación: "PDF descargado exitosamente" ✅

#### Características del PDF:
- ✅ **Formato A4** profesional
- ✅ **Alta calidad** (scale 2x)
- ✅ **Multi-página** si el contenido es largo
- ✅ **Exactamente** como lo ves en pantalla

---

### 9. **Ver Mis Documentos** 📂

1. **Navbar** → Click en **"Mis PDFs"**
2. Verás una **galería** de tus documentos guardados
3. Cada card muestra:
   - 📄 Nombre del documento
   - 📅 Fecha de última modificación
   - 🔢 Cantidad de secciones
4. Acciones disponibles:
   - **Editar** → Vuelve al editor
   - **🗑️** → Eliminar documento

---

### 10. **Explorar la Comunidad** 🌍

1. **Navbar** → Click en **"Comunidad"**
2. Verás la **Biblioteca Comunitaria** de secciones
3. **Tabs arriba** para filtrar por categoría
4. Cada sección muestra:
   - Nombre
   - Tipo
   - Preview del contenido
   - 🌐 Público / 🔒 Privado
5. Click en **"Usar en mi PDF"**
6. (Próximamente: Se agregará a tu biblioteca personal)

---

## 🎨 Tips Profesionales

### Tip #1: Workflow Rápido
```
1. Selecciona plantilla → 2. Edita textos → 3. Exporta
   (30 segundos)      (5 minutos)       (10 segundos)
```

### Tip #2: Colores Corporativos
Guarda tus colores favoritos:
- Verde A&T: `#1c5d15`
- Amarillo: `#e8ff99`
- Verde oscuro: `#0d350b`

### Tip #3: Tipografías Profesionales
Recomendadas:
- **Títulos**: Poppins Bold (700)
- **Cuerpo**: Poppins Regular (400)
- **Legal**: Roboto Regular

### Tip #4: Estructura Básica
```
┌─────────────┐
│ Hero        │ ← Título llamativo
├─────────────┤
│ About       │ ← Quiénes somos
├─────────────┤
│ Services    │ ← Qué ofrecemos
├─────────────┤
│ Values      │ ← Por qué elegirnos
├─────────────┤
│ Contact     │ ← Cómo contactarnos
└─────────────┘
```

### Tip #5: Antes de Exportar
✅ Revisa ortografía
✅ Verifica colores corporativos
✅ Comprueba que todo el texto sea legible
✅ Elimina secciones vacías

## ❓ Preguntas Frecuentes

### ¿Necesito internet?
No. Todo funciona **offline** con LocalStorage.

### ¿Dónde se guardan mis documentos?
En el **navegador** (LocalStorage). Si limpias el navegador, se pierden.
*Próximamente: En Supabase cloud.*

### ¿Puedo compartir mis PDFs?
Sí, exporta a PDF y comparte el archivo.

### ¿Las secciones que uso afectan a otros usuarios?
**No**. Cuando seleccionas una sección, se crea **TU COPIA**.
Modificas tu versión, no la original.

### ¿Puedo cambiar imágenes?
Por ahora no. Las imágenes vienen pre-definidas con Unsplash.
*Próximamente: Subir imágenes custom.*

### ¿Puedo crear mis propias secciones?
Sí, en **"Comunidad"** → **"Crear Nueva Sección"**.
*(Botón próximamente activado)*

### ¿Cuántas secciones puedo tener en un PDF?
**Ilimitadas**. Pero recuerda que se exporta a PDF, así que considera el tamaño.

## 🎓 Ejemplo Completo

### Caso de Uso: Brochure para Empresa de Tecnología

**Paso 1:** Registrarse
```
Email: ejemplo@empresa.com
Contraseña: ********
```

**Paso 2:** Usar plantilla "A&T BioNano"
```
Click en Templates → A&T BioNano → Usar Plantilla
```

**Paso 3:** Personalizar textos
```
Hero:
  Título: "TechCorp Solutions"
  Subtítulo: "Innovación en Software Empresarial"

About:
  Texto: "Somos líderes en desarrollo de software..."
```

**Paso 4:** Cambiar colores
```
Click en título → Color picker → #0066CC (azul corporativo)
```

**Paso 5:** Reordenar secciones
```
Arrastra "Servicios" antes de "Valores"
```

**Paso 6:** Agregar sección extra
```
Click + → Selecciona "Imagen + Texto" → Edita contenido
```

**Paso 7:** Guardar
```
Nombre: "Brochure TechCorp 2026"
```

**Paso 8:** Exportar
```
Click "Exportar PDF" → Descarga "Brochure TechCorp 2026.pdf"
```

**Tiempo total:** ~10 minutos ⏱️
